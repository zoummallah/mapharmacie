Projet Gestion Pharmacie
Installation :
utiliser le git pour le télécharger le dossier gestion_pharmacie
Copiez ce dossier dans votre répertoire htdocs de XAMPP .
Importez le fichier base_de_donnees.sql dans votre phpMyAdmin.
Lancez Apache et MySQL dans XAMPP et accédez à localhost/gestion_pharmacie.
